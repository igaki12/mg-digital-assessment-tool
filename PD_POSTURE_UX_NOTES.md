# PD Posture and UX Notes

本メモは、PD 向け姿勢評価を追加する際の補足整理である。正式要件ではなく、`PRODUCT_REQUIREMENTS_MG_PD.md` を詳細化するための検討メモとして扱う。

## 1. 評価対象

### 1.1 体幹前屈角

- 想定状態: 立位 / 歩行時に体幹が前方へ大きく屈曲する
- 推奨撮影: 側面
- 主なランドマーク候補:
  - 肩
  - 股関節
  - 膝
- 補助指標:
  - 体幹軸の前傾角
  - 大腿に対する体幹角

### 1.2 首下がり角

- 想定状態: 頭部が前方へ垂れ下がる
- 推奨撮影: 側面
- 主なランドマーク候補:
  - 耳
  - 肩
  - 股関節
- 補助指標:
  - 頭頸部の前傾角
  - 体幹軸に対する頭部偏位

### 1.3 側方偏位

- 想定状態: 体幹が左右どちらかに傾く
- 推奨撮影: 正面
- 主なランドマーク候補:
  - 左右肩の中点
  - 左右股関節の中点
- 補助指標:
  - 正中線の鉛直からのずれ
  - 肩線の傾き

## 2. UI / UX の補足

- 正面撮影用と側面撮影用で別のシルエットガイドを出す
- 立ち位置と画角が適切かを色で返す
- 「もう少し下がる」「横を向く」など短い指示を大きく表示する
- 音声ガイドを前提にし、横向きでも操作できるようにする
- 自動開始より、まずは位置合わせ完了の通知を優先する

## 3. 実装上の注意

- 単眼カメラでは奥行き方向の姿勢誤差が残る
- 衣服、照明、カメラ高さで角度がぶれやすい
- 医学的診断名を推定する機能にはしない
- 角度の絶対値より、同一条件下での再現性を優先する

## 4. 次の設計タスク

- 疾患プロファイル別の測定メニュー定義
- 正面 / 側面の撮影ガイド UI 設計
- 保存スキーマへの `disease` と `task` の追加
- MG 用 UI と混同しない文言設計
